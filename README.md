# templ

## Features

Syntax highlighting for the templ language (https://github.com/a-h/templ).

## Requirements

Requires templ command line on the path.

## Issues

Please raise any issues in https://github.com/a-h/templ

## Tasks

### compile

```sh
npm run compile
```

### version

requires: compile

```sh
npm run version
```

### package

requires: version

```sh
npm run package
```

### publish

requires: package

```sh
npm run package
```
